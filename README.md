# 🎓 Student Management REST API

A simple RESTful API built with **Spring Boot** that allows users to manage student records.  
The application supports creating and retrieving students stored in a **PostgreSQL** database.

This project demonstrates basic backend development concepts such as REST APIs, database integration, and entity mapping using JPA.

---

## 🚀 Features

✅ Create a new student  
✅ Retrieve all students  
✅ Auto-increment student ID  
✅ RESTful API architecture  
✅ Database persistence  

---

## 🛠 Technologies Used

- ⚙️ Spring Boot
- 🌐 Spring Web
- 🗄 Spring Data JPA
- 🔄 Hibernate
- 🐘 PostgreSQL
- 📦 Maven
